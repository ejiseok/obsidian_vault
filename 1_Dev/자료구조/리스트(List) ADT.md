# 개념

- 목록 형태로 이루어진 데이터 형식
- 목록을 이루는 개별 요소를 **노드(Node)** 라고 부른다.
- 첫 번째 노드를 **헤드(Head)**, 마지막 노드를 **테일(Tail)** 이라고 부른다.

# 연산 종류

- 리스트에 노드를 추가하는 연산(Append)
- 노드 사이에 노드를 삽입하는 연산(Insert)
- 노드를 제거하는 연산(Remove)
- 특정 위치에 있는 노드를 반환하는 연산(Get At) 등

# 배열과의 차이점

- 배열은 생성하는 시점에 반드시 배열의 크기를 지정해줘야 하고 생성한 후 크기를 변경할 수 없는 단점이 있다.
- 리스트는 배열처럼 데이터의 집합을 보관하는 기능을 가지면서도 유연하게 크기를 바꿀 수 있는 [[자료구조(Data Structure)|자료구조]]다.
